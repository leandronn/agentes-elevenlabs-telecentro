# Demo estática Naranja X / ElevenLabs

Carpeta con 4 páginas HTML navegables entre sí:

- `index.html`
- `contacto.html`
- `prestamos.html`
- `ayuda.html`

## Uso

Abrir `index.html` en el navegador. No requiere build, Node, backend ni servidor.

## Importante

- Es una demo estática para presentación.
- No hay formularios reales ni envío de datos.
- Incluye `meta robots="noindex,nofollow"`.
- Los enlaces internos apuntan a los HTML locales.
- Las imágenes se cargan desde el CDN público de Naranja X / Contentful, según assets visibles en las páginas revisadas.
- Hay slots comentados para insertar ElevenLabs:
  - `ELEVENLABS_WIDGET_SLOT_HOME`
  - `ELEVENLABS_WIDGET_SLOT_CONTACTO`
  - `ELEVENLABS_WIDGET_SLOT_PRESTAMOS`
  - `ELEVENLABS_WIDGET_SLOT_AYUDA`

## Nota legal / marca

No se copió HTML, CSS ni JavaScript productivo del sitio original. El diseño fue recreado desde cero como mockup visual no funcional.
