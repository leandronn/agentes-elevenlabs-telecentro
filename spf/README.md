# DOS SPF static demo

Maqueta HTML estática generada para presentación interna.

## Archivos principales

- `index.html`
- `contacto.html`
- `sedes.html`
- `prestadores_externos.html`
- `farmacias.html`
- `vademecum.html`
- `noticias.html`
- `tramites.html`
- `turnos.html`

## Seguridad / alcance

- No hay backend.
- No hay login real.
- No se capturan credenciales ni datos personales.
- Los formularios son visuales y no consultan datos reales.
- Incluye `noindex,nofollow`.
- Las imágenes se referencian desde URLs públicas del dominio `www.obrasocial.spf.gob.ar/drive/repo/...`.

## Slots ElevenLabs

Buscá estos comentarios en los HTML y pegá el widget donde corresponda:

- `ELEVENLABS_WIDGET_SLOT_HOME`
- `ELEVENLABS_WIDGET_SLOT_CONTACTO`
- `ELEVENLABS_WIDGET_SLOT_PRESTADORES`
- `ELEVENLABS_WIDGET_SLOT_FARMACIAS`
- `ELEVENLABS_WIDGET_SLOT_VADEMECUM`
- `ELEVENLABS_WIDGET_SLOT_TURNOS`
- `ELEVENLABS_WIDGET_SLOT_TRAMITES`
- `ELEVENLABS_WIDGET_SLOT_NOTICIAS`
- `ELEVENLABS_WIDGET_SLOT_GLOBAL`

## Cómo abrir

Abrí `index.html` directamente en el navegador. Para que carguen las imágenes remotas, necesitás conexión a internet.
